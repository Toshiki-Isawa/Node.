Node. プライバシーポリシー

最終更新日: 2026年5月26日 · 施行日: 2026年5月26日

1. はじめに

Node.（以下「本アプリ」）は、植物の観測・成長記録を蓄積するための iOS アプリケーションです。本プライバシーポリシー（以下「本ポリシー」）は、本アプリの利用にあたり、当方がどのような情報を取得し、どのように利用・保管・共有するかを説明するものです。

本アプリを利用することにより、本ポリシーに同意したものとみなします。同意いただけない場合は、本アプリの利用を中止してください。

1.1 リリース段階について

現行の App Store 初版（v1.0）では、観測記録は端末内のみに保存され、当方サーバーには送信されません。有料クラウド同期（v1.1 以降）を利用する場合に、以下のクラウド関連条項が適用されます。

段階当方サーバーへの送信
v1.0（現行）なし
v1.1（予定）有料プラン（Archive / Conservatory）利用時

3. 取得する情報

3.0 v1.0（現行）で取得する情報

v1.0 では当方サーバーへのデータ送信はありません。植物・観測・育成ログ・タイムラプス MP4 は端末内のみに保存されます。アプリ削除・機種変更によりデータが失われる場合があります。

3.1 アカウント情報（v1.1 以降）

有料クラウド同期のためサインインした場合に、以下の情報を取得します。

情報取得元用途
ユーザー IDSupabase Auth（Sign in with Apple / Google Sign-In 経由）アカウント識別、データの紐付け
認証トークン同上セッション管理、クラウド同期
メールアドレス等Apple / Google（提供される場合のみ）認証、アカウント管理

Sign in with Apple では、Apple の設定に応じてメールアドレスが非公開（リレーアドレス）となる場合があります。

3.2 ユーザーが登録・記録するコンテンツ

情報内容保存先
植物情報植物名、種・品種、カテゴリ、取得日、水やり間隔端末内 / v1.1: Supabase
観測記録撮影写真、メモ、観測日時端末内 / v1.1: Supabase + R2
育成ログ水やり・施肥・植え替え等端末内 / v1.1: Supabase

観測写真には、植物そのもののほか、背景や撮影環境が写り込む場合があります。ユーザー自身の判断で記録内容を管理してください。

3.3 端末内のみで処理される情報

v1.0 の全記録、タイムラプス MP4、および v1.1 の R2 一時キャッシュは、原則として当方サーバーへ送信・保存しません。

情報内容
v1.0 の全記録植物・観測・育成ログ（端末内のみ）
タイムラプス動画端末内生成 MP4。クラウド非保存

3.4 課金情報（v1.1 以降）

情報取得元用途

決済処理そのものは Apple が行います。 当方がクレジットカード番号等の決済情報を直接取得することはありません。

3.5 分析・広告（v1.1 以降）

PostHog・AdMob は v1.0 では使用しません。v1.1 以降、Timelapse Export 時の広告等で利用する場合があります。

6. 情報の保存場所と保管期間

6.1 保存場所

データ種別保存場所
端末内データv1.0: 端末のみ / v1.1: 端末 + クラウド（有料時）
アカウント・観測画像v1.1 有料同期時: Supabase / R2

6.2 保管期間

  • アカウント存続中: 同期を有効にしている間、クラウド上のデータを保管します。
  • アカウント削除後: 当方サーバー上のデータは、削除処理完了後、合理的な期間内に削除します（完全削除まで最大 30 日程度を要する場合があります）。
  • 端末内データ: アプリ削除または端末内データの消去まで端末に残ります。アカウント削除だけでは、端末内のローカルデータは自動的には消去されません。

7. 情報の共有

当方は、以下の場合を除き、ユーザーの個人情報を第三者に販売または提供しません。

  1. 上記「第三者サービス」への業務委託
  2. 法令に基づく開示請求への対応
  3. 人の生命・身体・財産の保護に必要な場合
  4. 合併・事業譲渡等により事業が承継される場合

8. セキュリティ

  • Supabase Row Level Security（RLS）によるユーザー単位のアクセス制御
  • R2 への画像アップロード・ダウンロード時の presigned URL 利用
  • 通信の暗号化（HTTPS / TLS）
  • 認証情報の安全な管理

ただし、インターネット上の通信および電子保存の性質上、完全な安全性を保証するものではありません。

9. ユーザーの権利と選択肢

9.1 端末内での利用（v1.0）

v1.0 では記録は端末内のみです。アプリ削除・機種変更でデータを失う場合があります。

9.2 クラウド同期（v1.1 以降）

Archive / Conservatory プランでサインイン・クラウドバックアップが利用可能になります。

9.3 端末内データの削除

本アプリ内の記録削除機能、iOS の設定からのアプリデータ削除、またはアプリのアンインストールで消去できます。

9.4 トラッキングの拒否

App Tracking Transparency で拒否した場合でも、主要機能は利用できます。非パーソナライズ広告が表示される場合があります。

9.5 サブスクリプションの管理

iOS の「設定 > Apple ID > サブスクリプション」から解約・変更できます。

9.6 開示・訂正・削除等の請求

第 2 条のお問い合わせ先までご連絡ください。本人確認のうえ、法令に従い対応します。

10. 未成年者

本アプリは、13 歳未満（またはお住まいの地域で定められた最低年齢未満）の方を対象としていません。

11. 国外への情報移転

Supabase、Cloudflare、PostHog、Google 等は、日本国外にサーバーを置く場合があります。各サービス提供者のセキュリティ基準および適用法令に従って情報が処理されます。

12. 本ポリシーの変更

法令の改正、サービス内容の変更等に応じて、本ポリシーを改定することがあります。重要な変更がある場合は、本アプリ内または Web サイト上で告知し、改定後の施行日を明示します。
