## 1. はじめに

Node.（以下「本アプリ」）は、植物の観測・成長記録を蓄積するための iOS アプリケーションです。本プライバシーポリシー（以下「本ポリシー」）は、本アプリの利用にあたり、当方がどのような情報を取得し、どのように利用・保管・共有するかを説明するものです。

本アプリを利用することにより、本ポリシーに同意したものとみなします。同意いただけない場合は、本アプリの利用を中止してください。

## 3. 取得する情報

本アプリでは、機能の提供に必要な範囲で、以下の情報を取得・生成します。

### 3.1 アカウント情報

サインイン時に、以下の情報を取得します。

| 情報 | 取得元 | 用途 |
|------|--------|------|
| ユーザー ID | Supabase Auth（Sign in with Apple / Google Sign-In 経由） | アカウント識別、データの紐付け |
| 認証トークン | 同上 | セッション管理、クラウド同期 |
| メールアドレス等 | Apple / Google（提供される場合のみ） | 認証、アカウント管理 |

**Sign in with Apple** では、Apple の設定に応じてメールアドレスが非公開（リレーアドレス）となる場合があります。

### 3.2 ユーザーが登録・記録するコンテンツ

| 情報 | 内容 | 保存先 |
|------|------|--------|
| 植物情報 | 植物名、種・品種、カテゴリ、取得日、水やり間隔 | 端末内（SwiftData）／サインイン時は Supabase |
| 観測記録 | 撮影写真、メモ、観測日時 | 端末内／サインイン・同期時は Supabase + Cloudflare R2 |
| 育成ログ | 水やり・施肥・植え替え等の種別、メモ、記録日時 | 端末内／サインイン時は Supabase |

観測写真には、植物そのもののほか、背景や撮影環境が写り込む場合があります。ユーザー自身の判断で記録内容を管理してください。

### 3.3 端末内のみで処理される情報

以下は、原則として当方のサーバーへ送信・保存しません。

| 情報 | 内容 |
|------|------|
| タイムラプス動画 | 観測写真から端末内（AVFoundation）で生成する MP4。クラウドには保存しません |
| オフライン利用時のデータ | サインイン前、またはオフライン時に端末内のみに保存される植物・観測・ログデータ |
| 一時キャッシュ | Compare / Timelapse 利用時に R2 から取得した画像の端末内キャッシュ |

### 3.4 課金・サブスクリプション情報

有料プラン（Archive / Conservatory）をご利用の場合、以下を取得・保存します。

| 情報 | 取得元 | 用途 |
|------|--------|------|
| プラン種別 | StoreKit 2 / 当方サーバー | 機能・容量の提供 |
| トランザクション ID | StoreKit 2 | 購入の検証、復元、不正防止 |
| 有効期限 | StoreKit 2 | サブスクリプション管理 |

**決済処理そのものは Apple が行います。** 当方がクレジットカード番号等の決済情報を直接取得することはありません。

### 3.5 利用状況・分析情報

PostHog を用いて、以下のイベントを送信する場合があります（API キー未設定時は送信しません）。

| イベント例 | 送信内容 |
|------------|----------|
| リワード広告の読み込み・表示・完了・失敗 | プラン種別、リトライ回数、エラーコード |

本分析は、主に Timelapse Export に伴う広告機能の改善を目的としています。植物の観測内容そのものは PostHog へ送信しません。

### 3.6 広告関連情報

Seed（無料）プランで Timelapse を書き出す際、Google AdMob のリワード広告を表示します。この際、以下が発生する場合があります。

| 項目 | 内容 |
|------|------|
| 広告識別子 | IDFA 等（App Tracking Transparency で許可された場合） |
| デバイス情報 | OS バージョン、端末モデル、IP アドレス等（AdMob が収集） |
| 同意情報 | Google User Messaging Platform（UMP）による広告パーソナライズ同意 |

Archive / Conservatory プランでは、広告 SDK を呼び出しません。

### 3.7 端末の権限・アクセス

本アプリは、以下の端末機能へのアクセスを求めます。

| 権限 | 目的 |
|------|------|
| カメラ | 観測写真の撮影 |
| 写真ライブラリ（読み取り） | 観測写真の保存・共有 |
| 写真ライブラリ（追加） | タイムラプス動画の保存 |
| App Tracking Transparency | より関連性の高い広告表示（Seed プランの Export 時。許可は任意） |

## 4. 情報の利用目的

取得した情報は、以下の目的で利用します。

1. 本アプリの提供（観測記録、タイムライン、Compare、Timelapse、Quick Log 等）
2. クラウド同期およびデータのバックアップ（サインイン時）
3. サブスクリプションの提供、検証、復元
4. プランに応じた保存品質・容量の管理
5. Seed プランにおける Timelapse Export 時のリワード広告配信
6. 広告・分析機能の改善および障害対応
7. お問い合わせへの対応
8. 法令遵守、利用規約違反への対応

## 5. 第三者サービスへの提供

本アプリは、サービス提供のため、以下の第三者サービスを利用します。各サービスは、それぞれのプライバシーポリシーに基づき情報を処理します。

| サービス | 提供元 | 用途 | 主な送信情報 |
|----------|--------|------|--------------|
| Supabase | Supabase, Inc. | 認証、データベース | ユーザー ID、植物・観測・ログのメタデータ |
| Cloudflare R2 | Cloudflare, Inc. | 観測画像のクラウド保存 | 画像ファイル、オブジェクトキー |
| Sign in with Apple | Apple Inc. | 認証 | Apple ID トークン |

| Google AdMob | Google LLC | リワード広告（Seed のみ） | デバイス識別子、広告関連データ |
| PostHog | PostHog, Inc. | 利用状況分析 | 広告関連イベント、プラン種別 |

## 6. 情報の保存場所と保管期間

### 6.1 保存場所

| データ種別 | 保存場所 |
|------------|----------|
| 端末内データ | お使いの iOS 端末 |
| アカウント・メタデータ | Supabase（ホスティングリージョンは Supabase プロジェクト設定に依存） |
| 観測画像 | Cloudflare R2 |
| 分析データ | PostHog（ホスト設定に依存。既定は US リージョン） |

### 6.2 保管期間

- **アカウント存続中**: 同期を有効にしている間、クラウド上のデータを保管します。
- **アカウント削除後**: 当方サーバー上のデータは、削除処理完了後、合理的な期間内に削除します（バックアップ等の技術的理由により、完全削除まで最大 30 日程度を要する場合があります）。
- **端末内データ**: アプリ削除または端末内データの消去まで端末に残ります。アカウント削除だけでは、端末内のローカルデータは自動的には消去されません。

## 7. 情報の共有

当方は、以下の場合を除き、ユーザーの個人情報を第三者に販売または提供しません。

1. 上記「第三者サービス」への業務委託
2. 法令に基づく開示請求への対応
3. 人の生命・身体・財産の保護に必要な場合
4. 合併・事業譲渡等により事業が承継される場合

## 8. セキュリティ

当方は、個人情報の漏えい、滅失、改ざん等を防止するため、以下を含む合理的な安全管理措置を講じます。

- Supabase Row Level Security（RLS）によるユーザー単位のアクセス制御
- R2 への画像アップロード・ダウンロード時の presigned URL 利用
- 通信の暗号化（HTTPS / TLS）
- 認証情報の安全な管理

ただし、インターネット上の通信および電子保存の性質上、完全な安全性を保証するものではありません。

## 9. ユーザーの権利と選択肢

### 9.1 サインインなしでの利用

本アプリは Local-first 設計のため、サインインしなくても端末内での観測・記録機能を利用できます。クラウド同期、課金、クラウドからの復元にはサインインが必要です。

### 9.2 アカウント削除

設定画面からアカウントを削除できます。削除時には、当方サーバー上の R2 オブジェクトおよび Supabase 上のアカウント・関連データを削除します。

### 9.3 端末内データの削除

端末内のデータを消去するには、本アプリ内の記録削除機能を利用するか、iOS の設定からアプリデータを削除、またはアプリをアンインストールしてください。

### 9.4 トラッキングの拒否

iOS の「App Tracking Transparency」ダイアログで拒否した場合でも、本アプリの主要機能は利用できます。トラッキング拒否時は、パーソナライズされた広告ではなく、非パーソナライズ広告が表示される場合があります。

### 9.5 サブスクリプションの管理

サブスクリプションの解約・変更は、iOS の「設定 > Apple ID > サブスクリプション」から行えます。

### 9.6 開示・訂正・削除等の請求

保有個人データの開示、訂正、利用停止、削除等を希望される場合は、第 2 条のお問い合わせ先までご連絡ください。本人確認のうえ、法令に従い対応します。

## 10. 未成年者

本アプリは、13 歳未満（またはお住まいの地域で定められた最低年齢未満）の方を対象としていません。当方が意図せず当該年齢未満の方の個人情報を取得したことが判明した場合、速やかに削除します。

---

## 11. 国外への情報移転

本アプリで利用する第三者サービス（Supabase、Cloudflare、PostHog、Google 等）は、日本国外にサーバーを置く場合があります。国外移転が行われる場合でも、各サービス提供者のセキュリティ基準および適用法令に従って情報が処理されます。

---

## 12. 本ポリシーの変更

当方は、法令の改正、サービス内容の変更等に応じて、本ポリシーを改定することがあります。重要な変更がある場合は、本アプリ内または Web サイト上で告知し、改定後の施行日を明示します。改定後に本アプリを継続利用した場合、改定後のポリシーに同意したものとみなします。
